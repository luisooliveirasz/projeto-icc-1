# Sistema de Controle de Estoque em C

Este projeto, feito para a disciplina de Introdução à Ciência de Computação I oferecida pelo ICMC-USP, implementa um sistema simples de gerenciamento de estoque e caixa utilizando a linguagem C.  
Ele permite cadastrar produtos, atualizar quantidades, alterar preços, realizar vendas e salvar os dados em arquivo binário.

---

## Funcionalidades

O sistema suporta os seguintes comandos:

### Inserir produto (`IP`)
Adiciona um novo produto ao estoque.

---

### Aumentar estoque (`AE`)
Aumenta a quantidade de um produto existente e desconta o custo do saldo.

---

### Modificar preço (`MP`)
Altera o preço de um produto existente.

---

### Vender produto (`VE`)
Realiza vendas de produtos até o código `-1` ser informado.

---

### Consultar estoque (`CE`)
Mostra todos os produtos cadastrados.

---

### Consultar saldo (`CS`)
Exibe o saldo atual do caixa.

---

### Finalizar sistema (`FE`)
Encerra o programa e salva automaticamente os dados.
